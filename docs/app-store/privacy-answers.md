# App Store Privacy Answers Draft

Use this as the starting point for App Store Connect privacy labels. Confirm with counsel before release.

Data collected:

- User ID: Chess.com username entered by the user. Used for app functionality. Not used for tracking.
- User Content: PGN text pasted or uploaded by the user. Used for app functionality. Not used for tracking.
- Gameplay Content: Public Chess.com games fetched for the entered username. Used for app functionality. Not used for tracking.

Data linked to user:

- Chess.com username and imported game data can be associated with the user's training profile inside the app.

Tracking:

- No third-party advertising.
- No cross-app or cross-site tracking.
- App Tracking Transparency prompt should not be required unless analytics or ads are added later.

Data storage:

- Stored locally on device through browser/local app storage.
- Clearable from the app menu.

Third-party data source:

- Chess.com public API.
