# App Review Notes

Pattern Coach analyzes public Chess.com games and pasted PGN files to create personalized chess training drills.

Reviewer flow:

1. Open the app.
2. Enter a public Chess.com username, such as `hikaru`, or choose "Explore with sample data".
3. Wait for public games to sync and analysis to complete.
4. Use Dashboard, Games, Mistake Lab, Drill Mode, and Analysis Board.

Important notes:

- No Chess.com password is requested or stored.
- Chess.com sync uses public game archive endpoints.
- No payment, gambling, contests, live-game assistance, ads, or cross-app tracking are included.
- The app stores profile settings and reports locally so the dashboard is available after reopening.
- Users can clear local data from the app menu.
